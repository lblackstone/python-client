Parameters:request_compaction (bool) – A boolean indicating whether or not to request compaction.
get_authenticated_connection(user, passwd, db='admin', ssl=True)¶

Get an authenticated connection to this instance.

Parameters:
  • user (str) – The username to use for authentication.
  • passwd (str) – The password to use for authentication.
  • db (str) – The name of the database to authenticate against. Defaults to 'Admin'.
  • ssl (bool) – Use SSL/TLS if available for this instance. Defaults to True.
Raises:

pymongo.errors.OperationFailure if authentication fails.

Parameters:ssl (bool) – Use SSL/TLS if available for this instance.
get_stepdown_window(*args, **kwargs)¶

Get information on this instance’s stepdown window.

set_stepdown_window(*args, **kwargs)¶

Set the stepdown window for this instance.

Date times are assumed to be UTC, so use UTC date times.

Parameters:
  • start (datetime.datetime) – The datetime which the stepdown window is to open.
  • end (datetime.datetime) – The datetime which the stepdown window is to close.
  • enabled (bool) – A boolean indicating whether or not stepdown is to be enabled.
  • scheduled (bool) – A boolean indicating whether or not to schedule stepdown.
  • weekly (bool) – A boolean indicating whether or not to schedule compaction weekly.
shards(*args, **kwargs)¶

Get a list of shards belonging to this instance.

Parameters:add_shard (bool) – A boolean indicating whether to add a new shard to the specified instance.
